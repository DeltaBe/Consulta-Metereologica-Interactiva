Meteo-Analytics Global
Sistema de Análisis Meteorológico e Inteligencia de Datos
📌 Descripción
Meteo-Analytics es una plataforma web interactiva diseñada para la visualización y análisis estadístico de datos climáticos históricos a nivel mundial. A diferencia de los sistemas estáticos, esta aplicación consume datos en tiempo real de la API Open-Meteo, permitiendo realizar estudios de temperatura de cualquier ciudad, país y año (desde 1940 hasta la actualidad).

Este proyecto fue desarrollado como parte del 8vo Cuatrimestre de Ingeniería, enfocándose en la integración de servicios cloud, procesamiento de datos con Python y generación de reportes profesionales.

🚀 Características Principales
Geocodificación Automática: Los usuarios pueden escribir nombres de ciudades y países; el sistema resuelve las coordenadas geográficas mediante una API de búsqueda.

Análisis Multidimensional:

Individual: Análisis detallado de temperaturas máximas y mínimas por día.

Tendencia: Evolución térmica mensual a lo largo de un año completo.

Comparativa: Superposición de métricas entre dos ciudades distintas para análisis regional.

KPIs Dinámicos: Cálculo instantáneo de Temperatura Máxima Absoluta, Mínima y Promedio.

Exportación Profesional: Generación de reportes en formato PDF que incluyen gráficos y resúmenes estadísticos.

Arquitectura Cloud-Ready: Optimizado para despliegue en Render sin dependencia de almacenamiento local persistente.
